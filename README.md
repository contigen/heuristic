# Heuristic - AI-Powered Code Documentation

![Heuristic Logo](./public/heuristic-logo.png)

## Overview

Heuristic is an intelligent documentation platform that leverages AI to analyze your code repositories and automatically generate comprehensive, insightful documentation. It reduces friction in onboarding experience and helps developers to understand codebases faster, improve code quality, and maintain better documentation with minimal effort.
